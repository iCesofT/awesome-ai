## 🎯 Cómo Usar

1. Navega a la fase que necesites según tu etapa en el ciclo de vida
2. Entra a la carpeta del skill específico
3. Lee el archivo `SKILL.md` para documentación completa
4. Utiliza los skills como referencia en tus agentes y workflows

### Casos de Uso Comunes

- **Planificación de arquitectura**: Comienza en `01-planning/`
- **Revisión de código**: Utiliza skills de `02-design/` y `03-development/`
- **Validación de calidad**: Consulta `04-testing/`
- **Despliegue seguro**: Aplica skills de `05-deployment/`
